阶段1：入门筑基（第1-2个月）
 
核心目标
 
建立网安底层认知，搞定所有必学基础概念，为后续认证和实战扫清障碍
 
学习内容（严格按笔记顺序）
补充：先看中科大网络基础（44小时）
 
1. Google Cybersecurity Certificate（谷歌网安证书）

- 学习顺序： Security基础  →  Network（网络概念+计算机原理）  →  数据库  →  云计算（AWS/Azure入门）  →  微软AD域认证基础 

- 重点掌握：OSI七层模型、TCP/IP协议栈、常见端口、常见攻击类型

2. Linux基础

- 同步刷TryHackMe  Pre-Security  路径 +  Linux Fundamentals  全3个模块

- 掌握：常用命令、文件权限、用户管理、日志查看

3. 蓝队前置知识

- NIST网络安全框架、风险评估与审计基础

- 工具入门：Wireshark抓包基础、Sysmon日志入门
 
验收标准
 
- 完成Google Cybersecurity Certificate全部8门课程

- 能独立完成Linux基本操作，看懂系统日志

- 能口述清楚常见网络协议和攻击原理
 
每日任务
 
- 2小时视频课程 + 1小时笔记整理 + 1小时实操

- 周末：复盘本周知识点，完成1套基础概念自测题
 
阶段2：行业敲门砖认证（第3-4.5个月）
 
核心目标
 
拿到全球通用的网安入门第一证，简历直接通过HR初筛
 
唯一目标：CompTIA Security+（SY0-701）
 
1. 学习资源

- 官方教材 + Professor Messer免费视频（B站有全集）

- 题库：Dion Training 6套题 + 官方模拟题

- 同步刷TryHackMe  CompTIA Security+  备考路径

2. 时间安排

- 第1-3周：过完整视频，梳理知识框架

- 第4-5周：刷第一遍题库，标记所有错题

- 第6周：二刷错题+模拟考试，稳定85分以上报名考试
 
验收标准
 
- 一次性通过CompTIA Security+考试

- 更新简历，添加「CompTIA Security+ 认证」
 
阶段3：SOC分析师实战（第5-7个月）
 
核心目标
 
从"懂理论"到"会干活"，掌握SOC分析师日常工作全流程，达到初级入职标准
 
学习内容（严格按笔记顺序）
 
1. SOC理论入门

- 课程：TryHackMe  SOC Level 1  路径 +  Cell One  完整实战

- 掌握：SIEM原理、日志分析、告警排查、事件响应流程

2. SOC模拟器实战（核心！）

- 按难度从低到高：

1.  Let's Defend （入门级，笔记标注"类似Cell One"）：完成所有入门级事件响应案例

2.  Cyber Defenders ：刷完前20个日志分析、恶意代码分析场景

3.  Hack The Box ： Starting Point  蓝队模块 +  SOC Challenges  专区

3. SOC专项认证

- 优先：TCM Security  SOC Analyst Certification （笔记标注"带整套蓝队人员技能"）

- 备选：LRC Mastery、CCPL1（网络安全防御初级认证）

4. 个人项目

- 搭建个人SIEM环境（ELK Stack + Sysmon）

- 模拟3个常见安全事件：暴力破解、钓鱼邮件、勒索软件感染，输出完整事件报告
 
验收标准
 
- 完成至少30个SOC模拟器实战案例

- 拿到1个SOC专项认证

- 能独立输出标准的安全事件分析报告
 
每日任务
 
- 1小时理论学习 + 3小时模拟器实战

- 每周输出1篇事件分析报告，存入GitHub作品集
 
阶段4：多方向进阶（第8-12个月）
 
核心目标
 
根据兴趣选择1-2个方向深耕，提升核心竞争力，拿到更高薪资
 
分支1：蓝队进阶（推荐，与SOC无缝衔接）
 
- 课程：Advanced Blue Teaming（高级蓝队）

- 认证：CCDL2（网络安全防御中级）、CDSA

- 技能：威胁狩猎、恶意代码分析、EDR/XDR使用、事件响应高级流程
 
分支2：云安全（当前最缺人，薪资最高）
 
- 微软Azure路线（优先）

1. SC-900（微软安全、合规与身份基础）→ 1周搞定（笔记标注）

2. AZ-500（Azure安全工程师助理）→ 1.5个月

- AWS路线

1. AWS Certified Cloud Practitioner（从业者认证）

2. AWS Certified Security - Specialty

- 实战：TryHackMe  Cloud Security  路径 + Hack The Box 云安全场景
 
分支3：渗透测试（红队，兴趣导向）
 
- 入门：eJPT（eLearnSecurity Junior Penetration Tester）

- 准备：TryHackMe  Complete Beginner  +  Penetration Testing  路径 +  PT1  模块（练习渗透测试概念）

- 实战：Hack The Box  Starting Point  +  Easy  难度机器

- 进阶：OSCP（行业黄金认证）

- 准备：TCM Security  Practical Penetration Testing (PTP)  课程

- 实战：Hack The Box  Medium  难度机器 +  OPTS  专区（笔记标注"难度够"）
 
分支4：GRC（合规与风险管理）
 
- 学习内容：ISO27001、等保2.0、GDPR、风险评估方法论

- 认证：CISA（信息系统审计师）、CISSP（工作3年后考）
